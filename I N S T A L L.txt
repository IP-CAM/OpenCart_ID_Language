IKUTI PETUNJUK SESUAI URUTAN DI BAWAH INI
1. INSTALL
2. SETTING ADMIN
3. SETTING HALAMAN

====================== 1. I N S T A L L =======================
- Copy folder indonesia yang ada di dalam folder indonesia_admin ke folder admin/language/
- Copy folder indonesia yang ada di dalam folder indonesia_catalog ke folder catalog/language/

Sehingga di dalam folder admin/language terdapat 2 folder:
- english
- indonesia

Sehingga di dalam folder catalog/language/terdapat 2 folder:
- english
- indonesia

- Copy folder system ke root (home) / ke tempat opencart anda install

====================== 2. Setting admin panel ==================
- Login ke Admin Panel, pilih System -> Localisation -> Language,
- Klik INSERT untuk membuat bahasa Indonesia. isikan parameter berikut:
* Language Name : Indonesia
* Code : id
* Locale : id_ID.UTF-8,id_ID,id-id,indonesia
* Image : id.png
* Directory : indonesia
* Filename : indonesia
* Status : Enable
* Sort Order : 2
-Kemudian Save

====================== 3. Setting Halaman ========================
- Pilih tab System -> Setting klik Edit, kemudian pilih tab "Local". set Language dan Administration Language ke indonesia
- Kemudian Save
